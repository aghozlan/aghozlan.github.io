Question V: Pure Geometry (4 pts)

Geometry Scheme Reference Figure

Part Detailed Working Points
1) Reproduce the figure: Accuracy of the drawn circle, tangents, and point projections. 0.50 pts
2) Calculate the length of [DE]:
(ED) is tangent to (C) at D ⇒ (OD) ⊥ (ED). Triangle ODE is right at D.
E is symmetric to A wrt B ⇒ AB = BE = 4 cm. OE = OB + BE = 2 + 4 = 6 cm. OD = radius = 2 cm.
By Pythagoras in ODE: DE2 + OD2 = OE2 ⇒ DE2 + 4 = 36 ⇒ DE = √32 = 4√2 cm. 		1.00 pt
3) a) Show Parallel lines:
(OD) ⊥ (ED) [Tangent property].
(FB) ⊥ (ED) [Given, F is orthogonal projection].
Two lines perpendicular to the same line are parallel. (DO) // (FB). 		0.75 pts
3) b) Calculate BF and EF using Thales:
In ΔODE, since (FB) // (OD), apply Thales's Theorem:
EF / ED = EB / EO = FB / OD.
EB / EO = 4 / 6 = 2/3.
EF = DE × (2/3) = 4√2 × (2/3) = (8√2) / 3 cm.
FB = OD × (2/3) = 2 × (2/3) = 4/3 cm. 		1.00 pt
4) Calculate tan(E):
In right triangle DEO, tan(E) = Opposite / Adjacent = OD / DE = 2 / 4√2 = 1 / 2√2 = √2 / 4. 		0.75 pts
