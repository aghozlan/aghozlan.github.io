Opacity Sorter

Objects interact with light differently. Sort each material into the correct category by tapping it then tapping the bin — or drag on desktop.

Opaque
No light passes through.
A complete shadow (umbra) forms behind it. Example: a brick wall, a metal plate.
Translucent
Some light passes through, but it scatters. You cannot see clearly through it. Example: frosted glass, tracing paper.
Transparent
Light passes through completely. You can see clearly through it. Example: clean glass, clear water.

E3 · Explain — Rectilinear Propagation

The Ray Tracer

Light travels in straight lines. This is called rectilinear propagation. Watch it happen: click anywhere on the canvas to place an opaque obstacle, then see how the shadow forms.

The Law
Light travels in straight lines in a uniform medium.
This is why: shadows have sharp edges · you cannot see around corners · the camera obscura works · eclipses are predictable.

E3 · Explain — Shadows

Shadow Builder

A point source creates one shadow. An extended source creates two: the umbra (total shadow) and the penumbra (partial shadow). Use the slider to change the source size and watch both regions form.

Umbra
Total shadow. A region where no light from the source reaches. Formed behind any opaque object blocking all light rays from an extended source.
Penumbra
Partial shadow. A region where some light from the source reaches. Surrounds the umbra. Gets larger as the source gets bigger.
