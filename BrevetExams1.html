Version A

Ex 1: 1. 12V = Nominal Voltage, 6W = Nominal Power. 2. \(I_n = P/U = 0.5\text{A}\). 3a. \(U_R = R \times I = 5\text{V}\). 3b. \(U_L = U_G - U_R = 12\text{V}\). 3c. Yes, \(U_L = \text{Nominal Voltage}\) (or \(I = I_n\)). 4. \(E = P \times t = 6 \times 1800 = 10,800\text{J}\).

Ex 2: 1. Alternating Sinusoidal. 2. \(U_m = 3 \text{ div} \times 5 \text{V/div} = 15\text{V}\). 3. \(U_{eff} = U_m / 1.4 = 10.7\text{V}\). 4. \(T = 4 \times 10 = 40\text{ms} = 0.04\text{s}\). 5. \(f = 1/T = 25\text{Hz}\).

Ex 3: 1. Two forces must have same line of action, opposite directions, equal magnitudes. 2. Weight (vertical, down), Tension (vertical, up). 3. \(W = m \times g = 2\text{N}\). 4. \(T = W = 2\text{N}\) (equilibrium). 5. \(\Delta L = T/k = 0.05\text{m}\). 6. \(L = L_0 + \Delta L = 20\text{cm}\).

Ex 4: Part A: 1. AC. 2. \(I = P/U = 10\text{A}\). 3. 15A fuse (safely above 10A). Part B: 1. Body A exerts force on B, B exerts equal/opposite force on A. 2. \(\vec{F}_{A/B} = -\vec{F}_{B/A}\).

Version B

Ex 1: 1. [Diagram checked]. 2. \(R_{eq} = (20 \times 30)/(20+30) = 12\ \Omega\). 3. \(I = U/R_{eq} = 1\text{A}\). 4. \(I_1 = 12/20 = 0.6\text{A}\), \(I_2 = 12/30 = 0.4\text{A}\). 5. \(I = I_1 + I_2\) (\(1 = 0.6 + 0.4\)).

Ex 2: 1. \(T = 1/f = 0.02\text{s}\). 2. \(U_m = 220 \times 1.41 = 310.2\text{V}\). 3. \(Y = U_m / S_v = 6 / 2 = 3 \text{ div}\). 4. Voltmeter in parallel, displays \(220\text{V}\).

Ex 3: 1. Direct proportionality. 2. \(T = k \times \Delta L\). 3. \(k = 2.5 / 0.05 = 50\text{N/m}\). 4. \(T = 50 \times 0.1 = 5\text{N}\).

Ex 4: Part A: 1. \(W = 5\text{N}\). 2. [Vector diagram drawn]. 3. \(\vec{N} + \vec{W} = \vec{0} \Rightarrow N = W = 5\text{N}\). Part B: 1. \(P = U \times I = 1100\text{W}\). 2. \(E = 1.1\text{kW} \times 2\text{h} = 2.2\text{kWh}\). 3. Cost = \(2.2 \times 1500 = 3300\text{ L.L.}\)
