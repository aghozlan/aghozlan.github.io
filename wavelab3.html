📋 How to Use the Simulation — Read Before Starting
▼
1
Select Oscillate at the top — this creates a continuous wave.
2
Set Tension to High — keep this for ALL investigations so results are consistent.
3
Set Damping to None — slider all the way left so the wave doesn't fade.
4
Select No End — the wave travels freely without reflecting.
5
Tick Rulers — a ruler appears. Drag it to measure wavelength (crest to crest).
6
Tick Timer — a stopwatch appears. Use it to time one full oscillation.
7
Press Pause ⏸ to freeze the wave — much easier to measure a still wave!
Frequency slider
Controls oscillations per second (Hz). Higher = more waves, shorter wavelength.
Amplitude slider
Controls wave height (cm). Does NOT affect speed — you will prove this!
Ruler tool
Drag to measure crest-to-crest distance = wavelength λ in cm.
Timer tool
Time one full up-down-up cycle of any dot = period T in seconds.
